Our Club

Del Norte's 3D Development Club is a great opportunity to broaden your horizons in the 3D modelling as well as game development field. We work with tools like Blender and Unity to create 3d models and import them as game assets into Unity. We then use coding software like Visual Studio to work on C# on Unity. We are excited to have new members!
